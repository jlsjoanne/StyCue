# Local Development Setup

This guide explains how to run the StyCue ASP.NET Core Web API locally on Windows or macOS.

## 1. Required tools

Install the following tools:

- .NET SDK 8 or later
- Git
- Visual Studio 2026 on Windows, or VS Code on macOS
- C# Dev Kit extension for VS Code if using macOS
- SQL tool:
  - Windows: SSMS
  - macOS: VS Code MSSQL extension

Optional:

- Azure CLI
- Docker Desktop

## 2. Clone the repository

```bash
git clone https://github.com/jlsjoanne/StyCue.git
cd stylecue
```
